# Proyecto de Sistemas de Gestión Empresarial

Este repositorio contiene un módulo personalizado desarrollado para **Odoo 17**, enfocado en la **gestión de inventario** y **mantenimientos** de las máquinas de la cafetería.

---

## Descripción del Proyecto

### Módulo Personalizado de Odoo 17

Este módulo proporciona herramientas para gestionar eficientemente el inventario de las máquinas de la cafetería, así como un sistema para llevar un control de los mantenimientos realizados y programados.

---

## Requisitos Previos

- **Odoo 17**: Asegúrate de tener instalado Odoo 17.

---

## Instalación del Módulo

1. **Clona el repositorio** utilizando el siguiente comando en tu terminal:

   ```bash
   git clone https://github.com/Code-Monkeys-SL/SistemasDeGestionEmpresarial.git
   ```

2. Coloca el módulo en el directorio de addons de Odoo.

3. Actualiza la lista de módulos en Odoo desde la interfaz de administración.

4. Instala el módulo a través del menú de aplicaciones.

## Funcionalidades

- **Gestión de Inventario**: Registro y seguimiento del estado de las máquinas.
- **Control de Mantenimientos**: Registro de mantenimientos realizados y fechas programadas para futuros mantenimientos.
